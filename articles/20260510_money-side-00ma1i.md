## なぜ月5万円なのか？

月5万円という金額には明確な理由があります。

### 現実的な目標金額

- **初心者でも到達可能**：スキルゼロからでも3〜6ヶ月で達成できる
- **生活への影響が大きい**：年間60万円の追加収入は生活の質を大きく変える
- **税金面でも比較的シンプル**：年間20万円以下なら確定申告不要（給与所得者の場合）

### 月5万円あれば何ができる？

- 貯金：年間60万円の貯蓄が可能
- 自己投資：スキルアップのための講座や書籍購入
- 生活費の補填：食費や娯楽費を気にせず使える
- 投資の原資：積立投資で将来の資産形成

## 副業選びの3つの基準

副業を選ぶ際は、以下の3つの基準を意識しましょう。

### 1. 初期投資が少ない

リスクを最小限に抑えるため、**初期投資が1万円以下**で始められるものを選びましょう。

**避けるべき副業例：**
- せどり（在庫リスク）
- フランチャイズ（高額な初期費用）
- 仮想通貨トレード（高リスク）

### 2. スキルが積み上がる

時給労働ではなく、**経験を積むほど収入が増える**副業を選びましょう。

**おすすめの特徴：**
- スキルが資産になる
- 単価が上がっていく
- ポートフォリオが構築できる

### 3. 自分の興味・強みと合致

継続できなければ意味がありません。**少しでも興味があり、得意分野に近い**ものを選びましょう。

## 収益化を加速させる5つのコツ

### 1. プロフィールを磨く

```markdown
【悪い例】
初心者ですが頑張ります！よろしくお願いします。

【良い例】
Web制作歴3ヶ月。HTML/CSS/JavaScriptを使用したレスポンシブサイト制作が可能です。
納期厳守・迅速なレスポンスを心がけています。
ポートフォリオ：https://...
【制作実績】
・コーポレートサイト（3件）
・ランディングページ（5件）
```

### 2. 提案文を工夫する

クライアントの課題を理解し、解決策を提示する提案文を作成しましょう。

**テンプレート：**
```
お世話になります。【あなたの名前】と申します。

【案件名】の件で応募させていただきました。

■ご提案内容
貴社の【課題】を【方法】で解決いたします。

■制作の流れ
1. ヒアリング（〜日）
2. デザイン案提示（〜日）
3. 修正対応（〜回まで無料）
4. 納品

■納期
ご発注から【日数】で納品可能です。

■実績
【関連する実績のURL】

ご検討のほど、よろしくお願いいたします。
```

### 3. 納期を守る

**最優先事項は納期厳守です。**信頼を失うと継続案件が取れません。

- 余裕を持ったスケジュール設定
- 進捗報告をこまめに行う
- 問題が発生したら即座に連絡

### 4. クライアントとの関係構築

- 丁寧なコミュニケーション
- レスポンスは24時間以内
- 期待を超える納品物を目指す
- アフターフォローも丁寧に

### 5. 継続案件を獲得する

新規営業は労力がかかります。**継続案件を1〜2つ持つことで、安定した収入**が得られます。

## やってはいけない5つのNG行動

### 1. 複数の副業に手を出す

一つのスキルを伸ばすことに集中しましょう。器用貧乏になると収入が伸びません。

### 2. 怪しい副業に手を出す

- 「1日5分で月30万円」などの誇大広告
- マルチ商法・ネットワークビジネス
- 高額情報商材の購入

### 3. 本業に支障をきたす

副業は本業があってこそ。睡眠時間を削りすぎないように注意しましょう。

### 4. 確定申告を怠る

年間20万円以上の所得があれば確定申告が必要です。

### 5. スキルアップを怠る

現状維持は衰退と同じ。常に学び続ける姿勢が重要です。

## 税金と確定申告の基礎知識

### 確定申告が必要なケース

**給与所得者の場合：**
- 副業の所得が年間20万円を超える場合

**計算方法：**
```
所得 = 収入 - 経費

例）
収入：年間60万円
経費：15万円（PC代、通信費、書籍代など）
所得：45万円 → 確定申告が必要
```

### 経費として計上できるもの

- パソコン・周辺機器
- インターネット通信費（按分）
- 書籍・教材費
- セミナー・講座受講費
- 家賃（在宅作業の場合、按分）
- 電気代（按分）

:::message alert
経費計上には領収書やレシートの保管が必須です。
会計ソフト（freee、マネーフォワードなど）の利用がおすすめです。
:::
