# はじめに

人生最大の買い物と言われる住宅購入。しかし、多くの方が初めての経験であり、「何をどう確認すればいいのか分からない」という不安を抱えています。

私は不動産業界で10年以上勤務し、数百組のお客様の住宅購入をサポートしてきました。その経験から言えるのは、**購入前の確認不足が後悔の9割を占める**ということです。

この記事では、住宅購入で絶対に押さえるべき7つのチェックポイントを、実例を交えながら解説します。これから住宅購入を検討している方、すでに物件探しを始めている方に、実践的な価値を提供できる内容となっています。

## 1. 予算設定：年収の何倍が適正なのか

### 一般的な目安と現実のギャップ

よく「住宅ローンは年収の5〜7倍まで」と言われますが、これはあくまで金融機関が貸してくれる上限であり、**返済可能額とは別**です。

**適正な借入額の計算式：**
```
年間返済額 ≤ 年収 × 25%（手取りベースでは30%以内）
```

### 実例：年収500万円の場合

```
年収500万円のAさんの場合：
- 借入可能額（銀行基準）：約3,500万円
- 適正借入額：約2,800万円
- 差額：700万円

月々の返済額の違い：
- 3,500万円借入：月10.7万円（35年、金利1.0%）
- 2,800万円借入：月8.5万円
→ 差額2.2万円/月 = 26.4万円/年
```

この差額は、修繕費、固定資産税、生活費の余裕に直結します。

### 見落としがちな追加コスト

住宅購入時には物件価格以外に以下のコストが発生します：

| 項目 | 目安金額 | タイミング |
|------|----------|------------|
| 仲介手数料 | 物件価格×3%+6万円+消費税 | 契約時 |
| 登記費用 | 30〜50万円 | 決済時 |
| 火災保険 | 20〜40万円（10年分） | 契約時 |
| 固定資産税 | 年10〜20万円 | 毎年 |
| 管理費・修繕積立金（マンション） | 月2〜4万円 | 毎月 |

**3,000万円の物件で、初期費用だけで約250〜300万円が必要**です。

## 2. 立地選定：資産価値を左右する最重要ポイント

### 駅徒歩15分の壁

不動産業界には「駅徒歩15分の壁」という言葉があります。データ分析からも明確です：

```
駅徒歩と資産価値の関係（築10年後の価格維持率）
- 徒歩5分以内：90〜95%
- 徒歩6〜10分：85〜90%
- 徒歩11〜15分：75〜85%
- 徒歩16分以上：60〜75%
```

### ハザードマップは必須確認

2019年の台風19号以降、ハザードマップの確認は必須になりました。

**確認すべき3つのハザードマップ：**

1. **洪水ハザードマップ**
   - 河川の氾濫リスク
   - 想定浸水深を確認

2. **土砂災害ハザードマップ**
   - 急傾斜地の崩壊リスク
   - 土石流の危険区域

3. **地震ハザードマップ**
   - 液状化リスク
   - 揺れやすさ

国土交通省の「ハザードマップポータルサイト」で全国の情報が確認できます：
https://disaportal.gsi.go.jp/

### 実際にあった失敗例

> Bさん（30代夫婦）は駅から徒歩20分の新築一戸建てを購入。
> 当初は「車があるから問題ない」と考えていましたが、
> 5年後の転勤で売却を試みたところ、購入価格の65%でしか売れず、
> 1,000万円以上の損失を出してしまいました。

## 3. 新築vs中古：それぞれのメリット・デメリット

### 価格面での比較

```
同じ立地・広さで比較した場合：
新築マンション：4,500万円
築10年中古：3,200万円（約30%安）
築20年中古：2,400万円（約47%安）
```

### 新築のメリット・デメリット

**メリット：**
- 設備が最新
- 10年間の瑕疵保証
- 住宅ローン減税の控除額が大きい
- 修繕の心配が当面不要

**デメリット：**
- 価格が高い（新築プレミアム約20%）
- 実物を見て買えない（未完成の場合）
- 購入直後から資産価値が下がる

### 中古のメリット・デメリット

**メリット：**
- 価格が安い
- 実物を確認できる
- 立地の選択肢が広い
- 管理状況が確認できる

**デメリット：**
- 設備が古い
- リフォーム費用が必要な場合がある
- 住宅ローン減税の控除額が少ない

### おすすめの選択基準

```
新築がおすすめ：
✓ 長期（30年以上）住む予定
✓ 予算に余裕がある
✓ 最新設備にこだわりたい

中古がおすすめ：
✓ 立地を最優先したい
✓ 初期費用を抑えたい
✓ リフォームでカスタマイズしたい
✓ 10〜15年で住み替える可能性がある
```

## 4. 内覧時の必須チェックリスト

### 物件を見るベストな時間帯

**平日の午前中＋休日の夕方、最低2回は訪問する**

- 平日午前：周辺環境の静けさ、日当たりを確認
- 休日夕方：騒音レベル、住民の雰囲気を確認

### マンション内覧チェックリスト

```markdown
□ 玄関・共用部
  □ エントランスの清掃状態
  □ 掲示板の内容（トラブル情報）
  □ 駐輪場・駐車場の使用状況
  □ ゴミ置き場の管理状態

□ 室内
  □ 水回りの水圧（実際に出してもらう）
  □ 扉・窓の開閉具合
  □ 収納の広さ（実測する）
  □ コンセントの位置と数
  □ 携帯電話の電波状況

□ 眺望・環境
  □ 日当たり（時間帯を変えて確認）
  □ 周辺の騒音レベル
  □ プライバシー（向かいの建物との距離）
  □ 将来の建築予定（空き地を確認）
```

### スマホアプリを活用した確認方法

**騒音測定：**
- アプリ「騒音測定器」（iOS/Android）
- 40dB以下：静か
- 50〜60dB：日常会話レベル
- 70dB以上：うるさい

**方位確認：**
- コンパスアプリで正確な方角を確認
- 「南向き」でも真南から30度以上ずれていることも

**間取り実測：**
- 「AR Ruler」などで実際の寸法を測定
- 図面と実際のサイズが異なるケースに対応

## 5. 住宅ローン選択：金利だけで選んではいけない

### 金利タイプの選び方

**変動金利：**
```
現在の金利：0.3〜0.5%
メリット：当面の返済額が少ない
リスク：将来金利上昇の可能性
向いている人：繰上返済できる余裕がある
```

**固定金利：**
```
現在の金利：1.3〜1.8%
メリット：返済額が確定、計画が立てやすい
デメリット：金利が高い
向いている人：安定志向、長期的な計画を重視
```

### 見落としがちな諸費用の差

同じ借入額3,000万円でも、銀行によって諸費用が大きく異なります：

```
A銀行（メガバンク）：
- 事務手数料：3.3万円
- 保証料：60万円
- 合計：約63万円

B銀行（ネット銀行）：
- 事務手数料：66万円（借入額の2.2%）
- 保証料：0円
- 合計：約66万円

C銀行（地方銀行）：
- 事務手数料：11万円
- 保証料：30万円
- 合計：約41万円
```

**初期費用だけで22万円の差が出ることも！**

### 団体信用生命保険の落とし穴

住宅ローンには団信（団体信用生命保険）が付帯しますが、**持病がある場合は注意が必要**です。

```
通常の団信：
- 加入できない場合、ローン自体が組めない

ワイド団信：
- 金利+0.3%で持病があっても加入可能
- 高血圧、糖尿病なども対象

無団信ローン：
- フラット35なら団信なしでも借入可能
- ただし別途生命保険の加入を推奨
```

## 6. 管理組合・修繕計画の確認（マンション）

### 修繕積立金の適正額

マンションは15年前後で大規模修繕が必要です。その資金が不足すると、**一時金の徴収や修繕積立金の大幅値上げ**が発生します。

**適正な修繕積立金の目安：**
```
専有面積70㎡のマンションの場合：
- 新築時：月8,000〜10,000円
- 10年後：月12,000〜15,000円
- 20年後：月18,000〜22,000円

段階的に増額される計画が健全
```

### 要注意な管理組合の特徴

```markdown
危険信号：
❌ 修繕積立金が相場より極端に安い
❌ 長期修繕計画が10年以上更新されていない
❌ 管理費の滞納が多い（5%以上）
❌ 理事会の議事録が公開されていない
❌ 大規模修繕の予定が決まっていない（築12年以上）
```

### 重要事項調査報告書の読み方

中古マンション購入時には「重要事項調査報告書」で以下を必ずチェック：

1. **修繕積立金の総額と戸数**
   - 総額 ÷ 戸数 ÷ 専有面積 = 1㎡あたりの積立額
   - 200円/㎡以上が目安

2. **管理費の滞納状況**
   - 全体の5%以上が滞納していたら要注意

3. **大規模修繕の履歴**
   - 過去の修繕内容と費用を確認

## 7. 契約前の最終確認：後戻りできないポイント

### 手付金を払う前に確認すべきこと

手付金（通常は物件価格の5〜10%）を支払うと、**買主都合でのキャンセルは手付金放棄**となります。

**契約直前の最終チェックリスト：**

```markdown
□ 重要事項説明書を事前に入手し、じっくり読む
□ 不明点はすべて質問し、納得する
□ 住宅ローンの事前審査が通っている
□ 家族全員が合意している
□ 勤務先の異動・転勤の可能性を確認
□ 物件を最低3回は見ている
□ 周辺環境を平日・休日に確認済み
□ ハザードマップを確認済み
□ 管理組合の状況を確認済み（マンション）
□ リフォーム費用の見積もりを取得（中古）
```
