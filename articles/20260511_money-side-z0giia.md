## はじめに：なぜ今、エンジニアの副業なのか

「本業だけの収入では不安」「スキルを活かして収入を増やしたい」――そんな思いを抱えるエンジニアの方は多いのではないでしょうか。

この記事では、**実務経験1年以上のエンジニアが副業で月5万円を安定して稼ぐ方法**を、実体験と具体的なステップを交えて解説します。私自身、本業と並行して副業を開始し、初月で3万円、3ヶ月目には月8万円を達成しました。

### この記事で得られること

- 副業案件の選び方と獲得方法
- 時給3,000円以上を実現する具体的戦略
- 本業と両立するための時間管理術
- 税金・確定申告の基礎知識
- よくある失敗パターンと回避方法

それでは、具体的に見ていきましょう。

## 副業の種類と収益性の比較

まず、エンジニアができる副業の種類を整理します。

### 1. クラウドソーシング（難易度：★☆☆）

**主なプラットフォーム**
- クラウドワークス
- ランサーズ
- ココナラ

**メリット**
- 案件数が豊富で始めやすい
- 小規模案件から挑戦できる
- 実績を積みやすい

**デメリット**
- 単価が低め（時給1,500〜2,500円程度）
- 手数料が高い（10〜20%）
- 競争が激しい

**おすすめの案件タイプ**
```
- WordPress構築・カスタマイズ：3〜10万円/件
- LP（ランディングページ）制作：5〜15万円/件
- 既存サイトの改修：2〜8万円/件
```

### 2. エージェント経由の案件（難易度：★★☆）

**主なサービス**
- ITプロパートナーズ
- Workship
- シューマツワーカー

**メリット**
- 単価が高い（時給3,000〜5,000円）
- 長期案件が多く安定収入
- 営業不要でエージェントが案件紹介

**デメリット**
- 実務経験2年以上が求められることが多い
- 週10〜20時間のコミットが必要
- 面談・選考プロセスがある

**月5万円達成の目安**
```
時給4,000円 × 週3時間 × 4週 = 48,000円
または
時給3,500円 × 週4時間 × 4週 = 56,000円
```

### 3. 技術ブログ・コンテンツ販売（難易度：★★★）

**収益化の方法**
- アフィリエイト
- note・Zenn Bookでの有料記事販売
- Udemyなどでの教材販売

**メリット**
- ストック型の収入
- 自分のペースで作業できる
- 個人ブランディングにつながる

**デメリット**
- 収益化まで時間がかかる（3〜6ヶ月）
- 継続的なコンテンツ制作が必要
- 安定収入にするのは難しい

## 月5万円を達成する具体的ステップ

### Step1：スキルの棚卸しと強みの明確化（1週目）

まずは自分の持っているスキルを整理しましょう。

**チェックリスト**
- [ ] 実務で使っている言語・フレームワーク
- [ ] 個人開発・ポートフォリオの有無
- [ ] 得意な開発領域（フロント/バック/インフラ等）
- [ ] 業務外で学習しているスキル
- [ ] コミュニケーション能力の自己評価

**強みの見つけ方**

```markdown
## 私の例

### できること
- React/Next.jsでのフロントエンド開発（実務2年）
- TypeScriptを使った型安全な開発
- レスポンシブデザインの実装
- REST API連携

### 差別化ポイント
- デザイナーとの協業経験が豊富
- Figmaからの実装が得意
- パフォーマンス最適化の知見
```

### Step2：プラットフォーム選択と登録（2週目）

初心者には**エージェント経由**をおすすめします。理由は以下の通りです。

1. **単価が高い**：クラウドソーシングの1.5〜2倍
2. **営業不要**：エージェントが案件を紹介
3. **トラブル対応**：契約関連をサポート

**登録すべきエージェント（優先順）**

1. **ITプロパートナーズ**
   - 週2日〜の案件が豊富
   - リモート案件多数
   - 登録はこちら（※アフィリエイトリンクではありません）

2. **Workship**
   - 時間単位の案件あり
   - スキルマッチング精度が高い

3. **シューマツワーカー**
   - 土日メインの案件
   - スタートアップ案件が多い

**プロフィール作成のコツ**

```markdown
# 良い例

## 職務経歴
株式会社○○（2021年4月〜現在）
- Next.js/TypeScriptを用いたtoC向けWebアプリケーション開発
- 月間100万PVのサービスで、コンポーネント設計から実装まで担当
- Storybookを導入し、デザインシステムを構築（開発効率30%向上）

## スキル
- JavaScript/TypeScript（実務2年）
- React/Next.js（実務2年）
- Git/GitHub（チーム開発経験あり）

## 稼働可能時間
- 平日夜：2〜3時間/日
- 土日：5〜8時間/日
- 合計：週15時間程度
```

### Step3：案件獲得と単価交渉（3〜4週目）

**面談で必ず確認すべきポイント**

- [ ] 具体的な作業内容と成果物
- [ ] 稼働時間の柔軟性（コアタイムの有無）
- [ ] コミュニケーション方法（Slack/週次MTG等）
- [ ] 契約期間と更新条件
- [ ] 支払いサイト（月末締め翌月末払い等）

**単価交渉のテクニック**

初回は相場通りでOKですが、2ヶ月目以降は交渉の余地があります。

```
【交渉例】
「現在の業務にも慣れてきて、〇〇の部分でも貢献できるようになりました。
次回更新のタイミングで、時給を3,500円→4,000円に見直していただくことは
可能でしょうか？」
```

**ポイント**
- 具体的な貢献実績を示す
- 相手にメリットを伝える
- 次回更新時など、タイミングを明示

### Step4：効率的な作業と時間管理（実践フェーズ）

副業で失敗する最大の理由は「時間管理」です。

**私が実践している時間術**

1. **平日のルーティン**
```
6:00 起床・朝食
7:00-8:00 副業作業（集中力が高い朝を活用）
8:30-18:00 本業
19:00-20:00 夕食・休憩
20:00-22:00 副業作業
22:30 就寝準備
```

2. **タスク管理**
   - Notionで案件ごとにタスクを管理
   - 1日の作業を15分単位で記録
   - 週次で振り返りと改善

3. **効率化ツール**
```javascript
// コード生成にGitHub Copilotを活用
// 例：React Componentのボイラープレート生成

// Before: 手動で10分かけて作成
// After: Copilotで1分で生成
```

**時間当たり生産性を上げるコツ**

- 同じ技術スタックの案件を選ぶ（学習コスト削減）
- テンプレート・ライブラリを整備
- 本業で得た知見を副業に活かす（相乗効果）

## 税金と確定申告の基礎知識

副業で年間20万円以上稼ぐと確定申告が必要です。

### 経費として計上できるもの

```
✅ 計上できる経費
- PC・モニター等の機器購入費（按分が必要）
- インターネット代（按分：副業使用時間の割合）
- 書籍・技術教材費
- セミナー・勉強会参加費
- コワーキングスペース利用料
- 電気代（按分が必要）

❌ 計上できない経費
- 通常の食費
- 本業の通勤費
- プライベートな買い物
```

### 青色申告のメリット

副業を事業所得として青色申告すると、最大65万円の控除を受けられます。

**条件**
- 開業届と青色申告承認申請書の提出
- 複式簿記での記帳
- e-Taxでの電子申告

**おすすめ会計ソフト**
- freee：初心者向け、自動仕訳が便利
- マネーフォワード：中級者向け、機能が豊富
- やよい：低価格、シンプル

### 住民税の注意点

副業収入があると住民税が増え、**本業の会社に副業がバレる**可能性があります。

**対策**
確定申告書の「住民税の徴収方法」で「自分で納付」を選択しましょう。

## よくある失敗パターンと対策

### 失敗1：低単価案件で消耗

**症状**
- 時給1,000円以下の案件を受けてしまう
- 納品後の修正が無限に続く

**対策**
- 最低でも時給2,500円以上の案件を選ぶ
- 契約前に修正回数の上限を明記
- 「この金額では厳しい」と断る勇気を持つ

### 失敗2：本業に支障が出る

**症状**
- 睡眠時間が4〜5時間になる
- 本業のパフォーマンスが低下

**対策**
- 週の作業時間を15時間以内に制限
- 納期に余裕のある案件を選ぶ
- 体調不良時は無理しない

### 失敗3：確定申告を忘れる

**症状**
- 確定申告の存在を知らず、後から税務署に指摘される

**対策**
- 収入・経費を毎月記録
- 会計ソフトで自動化
- 1月中に前年分の整理を完了

## 収入アップのための次のステップ

月5万円を達成したら、次のステップに進みましょう。

### 月10万円への道

**方法1：単価アップ**
- 実績を元に時給交渉
- より高度なスキルを習得（AWS、Dockerなど）
- 上流工程（設計・要件定義）にも関わる

**方法2：作業時間を増やす**
- 週15時間→20時間に拡大
- ただし、本業・健康とのバランスに注意

**方法3：複数案件の掛け持ち**
- メイン案件：週10時間
- サブ案件：週5〜8時間
- リスク分散にもなる

### 月20万円以上を目指すなら

この段階では「副業」ではなく「複業」「フリーランス」への転換を検討しましょう。

**選択肢**
1. フリーランスとして独立
2. 本業を週3〜4日勤務に変更（可能な会社なら）
3. 副業収入を元手に起業・サービス開発
