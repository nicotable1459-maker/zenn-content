# はじめに

人生で最も大きな買い物の一つである「住宅購入」。多くの人にとって初めての経験であり、知識不足から後悔するケースも少なくありません。

本記事では、住宅購入を検討している方に向けて、**購入前に知っておくべき重要ポイント**と**実践的なチェックリスト**を紹介します。これから住宅購入を考えている方、情報収集を始めたばかりの方に、実際の購入プロセスで役立つ具体的な知識を提供します。

:::message
この記事で得られること:
- 住宅購入の全体像と流れの理解
- 資金計画の立て方と注意点
- 物件選びの具体的なチェックポイント
- 契約時の重要確認事項
- 購入後のトラブル回避策
:::

## 住宅購入の全体像を把握する

### 購入までのステップ

住宅購入は以下の6つのステップで進みます。

1. **情報収集・予算決定**（1〜3ヶ月）
2. **物件探し**（2〜6ヶ月）
3. **物件見学・絞り込み**（1〜3ヶ月）
4. **購入申込・住宅ローン事前審査**（1〜2週間）
5. **売買契約・住宅ローン本審査**（1〜2ヶ月）
6. **決済・引き渡し**（1〜2ヶ月）

全体で**最短6ヶ月、通常1年程度**かかると考えておきましょう。

### 新築vs中古:それぞれのメリット・デメリット

| 項目 | 新築 | 中古 |
|------|------|------|
| 価格 | 高い（約3,500〜5,000万円）| 比較的安い（約2,500〜4,000万円）|
| 状態 | 最新設備・きれい | 築年数により差がある |
| 立地 | 郊外が多い | 駅近など好立地が多い |
| カスタマイズ | 間取り変更可能な場合も | リノベーション前提 |
| 税制優遇 | 手厚い | 条件により制限あり |
| 修繕費 | 当面不要 | 早期に必要な場合も |

**私の推奨**:
- **立地重視・予算抑えたい**→ 中古+リノベーション
- **新しさ・保証重視**→ 新築
- **資産価値重視**→ 駅近の中古マンション

## 資金計画:失敗しないための5つのルール

### ルール1:年収の5〜6倍が適正価格

```
例）年収600万円の場合
適正購入価格:3,000〜3,600万円
```

よく「年収の7倍まで」と言われますが、**余裕を持って5〜6倍**に抑えるのが安全です。

### ルール2:頭金は物件価格の20%以上を目指す

```
物件価格:3,500万円の場合
理想的な頭金:700万円（20%）
最低ライン:350万円（10%）
```

頭金が多いほど:
- 住宅ローンの金利が低くなる可能性
- 月々の返済額が減る
- 審査に通りやすくなる

### ルール3:諸費用は物件価格の5〜10%を別途用意

多くの人が見落とすのが**諸費用**です。

**新築マンションの諸費用例（3,500万円の物件）**
- 仲介手数料:0円（売主直販の場合）
- 登記費用:30〜50万円
- 住宅ローン手数料:70万円（借入額の2%程度）
- 火災保険・地震保険:20〜30万円
- 固定資産税・都市計画税日割り:5〜10万円
- 修繕積立基金:30〜50万円
- 管理費等前払い金:5〜10万円
- 引越し・家具購入費:50〜100万円

**合計:210〜320万円**

:::message alert
頭金とは別に諸費用分の貯金が必要です！
:::

### ルール4:月々の返済額は手取り月収の25%以内

```
手取り月収40万円の場合
適正返済額:10万円以内
```

**返済シミュレーション例**
```
借入額:3,000万円
金利:1.0%（全期間固定）
返済期間:35年

→ 月々返済額:約84,685円
→ 総返済額:約35,567,000円
→ 利息総額:約5,567,000円
```

### ルール5:変動金利のリスクを理解する

| 金利タイプ | 現在の相場 | メリット | デメリット |
|-----------|-----------|---------|-----------|
| 変動金利 | 0.3〜0.5% | 金利が低い | 金利上昇リスク |
| 固定金利(10年) | 1.0〜1.5% | 一定期間安定 | 期間後に再選択 |
| 全期間固定 | 1.5〜2.0% | 完済まで安定 | 金利が高い |

**私の推奨**:
- **安定志向・長期計画**→ 全期間固定
- **繰上返済予定あり**→ 変動金利
- **バランス型**→ 10年固定

## 物件選びの実践チェックリスト

### 立地条件のチェックポイント

#### 必須確認項目

- [ ] **駅徒歩時間**:実際に歩いて確認（不動産表記は80m=1分換算で実態と異なる場合も）
- [ ] **通勤時間**:ラッシュ時に実際に移動してみる
- [ ] **周辺環境**:スーパー、病院、学校、公園の位置
- [ ] **治安**:警視庁の犯罪マップで確認
- [ ] **ハザードマップ**:洪水・土砂災害・地震のリスクを確認

#### 将来性のチェック

```
良い例:
✓ 再開発計画がある
✓ 大学や企業の移転予定がある
✓ 新駅・新路線の開業予定

悪い例:
✗ 人口減少地域
✗ 商店街がシャッター街化
✗ 大型工場の閉鎖予定
```

### 建物・設備のチェックポイント

#### マンションの場合

**共用部分**
- [ ] エントランス・廊下の清掃状態
- [ ] 掲示板の管理状況（適切に更新されているか）
- [ ] 駐輪場・駐車場の整頓状態
- [ ] ゴミ置き場の清潔さ
- [ ] 共用施設（集会室等）の維持管理

**専有部分**
- [ ] 日当たり（南向きベストだが東南・南西も可）
- [ ] 眺望・視線（隣接建物から覗かれないか）
- [ ] 騒音（上下左右の部屋、道路、線路）
- [ ] 収納スペース（生活に必要な量を確保できるか）
- [ ] コンセント位置・数
- [ ] 携帯電波の入り具合

#### 一戸建ての場合

**構造・基礎**
- [ ] 基礎の種類（ベタ基礎が理想）
- [ ] 外壁のひび割れ有無
- [ ] 屋根の状態
- [ ] 床の傾き・軋み
- [ ] 雨漏りの痕跡

**土地**
- [ ] 境界の明確化（境界標の確認）
- [ ] 接道義務（幅4m以上の道路に2m以上接しているか）
- [ ] 地盤の強度
- [ ] 日照・風通し
- [ ] 高低差

### 管理組合・修繕計画のチェック（マンション）

これは**最重要**ですが見落とされがちです。

#### 重要確認事項

```
□ 修繕積立金の残高
  → 戸数×築年数×10万円以上が目安

□ 大規模修繕の実施履歴
  → 12〜15年ごとに実施が理想

□ 長期修繕計画の有無
  → 25〜30年の計画があるか

□ 管理費・修繕積立金の値上げ予定
  → 段階的に上がる計画が一般的

□ 管理会社の評判
  → ネット検索で悪評がないか確認

□ 総会の議事録
  → トラブル・問題の有無を確認
```

**危険サイン**
- 修繕積立金が著しく不足している
- 大規模修繕が延期されている
- 管理費滞納が多い
- 総会が成立しない（無関心な住民が多い）

## 契約時の重要確認事項

### 重要事項説明で必ず確認すべき10項目

重要事項説明は**契約前の最後の防衛ライン**です。

1. **物件の権利関係**:所有権に制限がないか
2. **法令上の制限**:建て替え時の制限など
3. **インフラ整備**:水道・ガス・電気の引き込み状況
4. **契約解除**:手付解除の期日、違約金の額
5. **ローン特約**:ローンが通らなかった場合の取り決め
6. **瑕疵担保責任**:欠陥が見つかった場合の責任範囲
7. **付帯設備**:エアコン・照明等何が含まれるか
8. **修繕履歴**:過去の修繕内容（中古の場合）
9. **近隣トラブル**:告知事項の有無
10. **引き渡し時期**:遅延時のペナルティ

:::message alert
重要事項説明は**録音を強く推奨**します。後で「聞いていない」とトラブルになることを防げます。
:::

### 住宅ローン審査のポイント

#### 事前に準備すべき書類

```
□ 本人確認書類（運転免許証等）
□ 収入証明書（源泉徴収票・確定申告書）
□ 勤続年数証明（健康保険証等）
□ 物件資料（販売図面、間取り図）
□ 借入状況（他のローン・クレジットカード）
```

#### 審査で見られるポイント

| 項目 | 基準 | 対策 |
|------|------|------|
| 年収 | 安定した収入 | 転職は審査後に |
| 勤続年数 | 3年以上が理想 | 最低1年は勤務 |
| 他の借入 | 総額が少ない | 可能なら完済 |
| 信用情報 | 延滞歴なし | 支払い遅延厳禁 |
| 健康状態 | 団信加入可能 | 持病は要相談 |

## 購入後のトラブル回避術

### 引き渡し時の徹底チェック

引き渡し当日は**内覧会**として2〜3時間かけて確認しましょう。

**チェックリスト（抜粋）**
```
□ 傷・汚れの有無（写真撮影必須）
□ ドア・窓の開閉
□ 水回りの動作確認（水を流す）
□ コンセント・照明の動作
□ エアコンの動作（設置されている場合）
□ 床の傾き（ビー玉を転がす）
□ 設備の取扱説明書・保証書
□ 鍵の本数確認
□ 付帯設備の動作確認
```

可能なら**専門家（ホームインスペクター）**の同行を推奨します（費用:5〜10万円）。

### 住み始めてからの注意点

#### 近隣への挨拶

```
挨拶範囲:
- 一戸建て:両隣・向かい3軒・裏3軒
- マンション:上下・両隣

手土産:500〜1,000円程度の日用品
（タオル・洗剤・お菓子など）
```

#### 住宅ローン控除の確定申告

初年度は**確実に確定申告**してください（2年目以降は年末調整で可）。

**必要書類**
- 住宅ローンの年末残高証明書
- 売買契約書のコピー
- 登記事項証明書
