## はじめに：カスタム指示が変えるAI活用の世界

ChatGPTを使っていて「毎回同じような指示を繰り返している」「もっと自分好みの回答がほしい」と感じたことはありませんか？

**カスタム指示（Custom Instructions）**は、ChatGPTに対してあなたの情報や希望する回答スタイルを事前に登録できる機能です。これを設定することで、毎回の会話で背景説明をする手間が省け、より的確で個別最適化された回答を得られるようになります。

本記事では、実務ですぐに使える具体的なカスタム指示の設定例を15個紹介します。コピペして使えるテンプレート付きなので、明日からすぐに活用できます。

- カスタム指示の基本的な使い方

- 効果を最大化するための書き方のコツ
- よくある失敗例と改善方法

## カスタム指示とは？基本の2つの質問

カスタム指示は、以下の2つの項目で構成されています：

### 1. あなた自身について（What would you like ChatGPT to know about you?）

- あなたの職業、専門分野
- 現在取り組んでいるプロジェクト
- スキルレベルや知識の程度
- 重視している価値観

### 2. ChatGPTにどのように応答してほしいか（How would you like ChatGPT to respond?）

- 回答の長さや形式
- トーンやスタイル
- 避けてほしい表現
- 含めてほしい要素

## 設定方法（3ステップ）

1. ChatGPTの画面右下のプロフィールアイコンをクリック
2. 「Settings」→「Personalization」→「Custom instructions」を選択
3. 2つの質問に回答して「Save」

:::message
※カスタム指示機能は有料プラン（Plus, Team, Enterprise）で利用可能です
:::

## カスタム指示を効果的にするための5つのコツ

### 1. 具体的に書く

❌ 悪い例：「分かりやすく説明してください」
⭕ 良い例：「中学生にも理解できるように、専門用語を使わず具体例を3つ以上含めて説明してください」

### 2. 制約条件を明示する

あなたの制約（時間、予算、スキルレベルなど）を明記することで、現実的な提案が得られます。

```
予算：月10万円以内
実装期間：2週間
チーム人数：3名
```

### 3. 出力フォーマットを指定する

```
【応答スタイル】
- 箇条書きで5項目以内にまとめてください
- 各項目は「結論→理由→具体例」の順で構成してください
- 最後に3行でサマリーをつけてください
```

### 4. 避けてほしいことも書く

```
- 一般論だけで終わらせないでください
- 「〜だと思います」のような曖昧な表現は避けてください
- 実行不可能な理想論は除外してください
```

### 5. 定期的に見直す

あなたの状況や目的は変化します。3ヶ月に一度は設定を見直しましょう。
