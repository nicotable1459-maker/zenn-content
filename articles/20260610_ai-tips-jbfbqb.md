## ChatGPT活用の3つの鉄則

### 1. 具体的な文脈を提供する

曖昧な質問ではなく、環境・制約・目的を明確に伝えることで、より実用的な回答が得られます。

**悪い例:**
```
エラーが出ます。直し方を教えてください。
```

**良い例:**
```
Next.js 14でビルド時に以下のエラーが発生します。
環境: Node 18, TypeScript 5.2
エラー: [具体的なエラーメッセージ]
該当コード: [コードスニペット]
試したこと: [これまでの対処法]
```

### 2. 出力形式を指定する

「表形式で」「コードブロックで」「チェックリストで」など、欲しい形式を明示すると、そのまま使える出力が得られます。

### 3. 批判的に検証する

ChatGPTの出力は必ず検証しましょう。特にセキュリティ関連やパフォーマンスが重要な部分では、テストやレビューを怠らないことが大切です。
