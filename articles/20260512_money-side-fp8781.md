## 副業収入と確定申告の基本

### 確定申告が必要になる条件

副業収入がある場合、以下の条件で確定申告が必要になります:

1. **給与所得者の場合**: 副業の所得が年間20万円を超える
2. **個人事業主の場合**: 所得が基礎控除額(48万円)を超える

```
副業所得の計算式:
所得 = 収入 - 必要経費

例)
ブログ広告収入: 50万円
必要経費(サーバー代、ドメイン代など): 8万円
→ 所得 = 42万円(確定申告が必要)
```

### 雑所得 vs 事業所得の違い

この区別は非常に重要です:

| 項目 | 雑所得 | 事業所得 |
|------|--------|----------|
| 青色申告 | 不可 | 可能(最大65万円控除) |
| 損失の繰越 | 不可 | 3年間可能 |
| 事業専従者給与 | 不可 | 可能 |
| 判断基準 | 副次的・一時的 | 継続的・反復的 |

**事業所得と認められるポイント:**
- 継続して反復的に行っている
- 相当な時間と労力を投入している
- 安定した収入がある
- 事業として社会的に認知される形態

#### 物販・ハンドメイド副業の場合

**按分の具体例:**

```
自宅で副業をしている場合の家賃按分:

家賃: 月10万円(年間120万円)
作業スペース: 6畳
家全体: 30畳

業務使用割合 = 6畳 ÷ 30畳 = 20%
経費計上可能額 = 120万円 × 20% = 24万円/年
```

### 2. 青色申告の活用

青色申告は副業を事業所得として認められた場合に選択できます。

**青色申告のメリット:**

1. **青色申告特別控除**: 最大65万円の控除
   - e-Taxでの電子申告: 65万円控除
   - 紙での申告: 55万円控除
   - 簡易簿記: 10万円控除

2. **青色事業専従者給与**: 家族への給与を経費にできる

3. **純損失の繰越控除**: 赤字を3年間繰り越せる

**青色申告の始め方:**

```
タイムライン:
3月15日まで: 前年分の確定申告
3月15日まで: 青色申告承認申請書を提出
(新規開業の場合は開業から2ヶ月以内)

必要書類:
- 青色申告承認申請書
- 開業届(個人事業の開業・廃業等届出書)
```

### 3. 小規模企業共済の活用

副業が軌道に乗ってきたら検討したいのが小規模企業共済です。

**メリット:**
- 掛金が全額所得控除(月額1,000円〜70,000円)
- 退職金・年金として受け取れる
- 低金利で貸付制度も利用可能

**節税効果の試算:**

```
年収500万円の会社員が副業で年間100万円の所得がある場合:

小規模企業共済に月5万円(年60万円)を拠出
→ 約18万円の税金が軽減される計算
(所得税・住民税合わせて約30%の税率の場合)
```

### 4. iDeCoとの併用戦略

iDeCo(個人型確定拠出年金)も全額所得控除の対象です。

**併用のポイント:**

| 制度 | 小規模企業共済 | iDeCo |
|------|--------------|-------|
| 掛金上限 | 月7万円 | 月1.2万〜6.8万円 |
| 受取時の税制 | 退職所得控除 | 退職所得控除・公的年金控除 |
| 途中解約 | 可能(元本割れリスク有) | 原則60歳まで不可 |

**おすすめの併用パターン:**

```
会社員(企業年金なし)の場合:
- iDeCo: 月2.3万円(年27.6万円)
- 小規模企業共済: 月3万円(年36万円)
合計: 年63.6万円の所得控除
```

### 5. ふるさと納税の上限最適化

副業収入があると、ふるさと納税の上限額が増えます。

**上限額の計算:**

```
目安の計算式:
ふるさと納税上限額 ≒ (住民税所得割額 × 20%) ÷ (100% - 住民税率 - 所得税率) + 2,000円

簡易シミュレーション:
給与収入500万円のみ: 上限約6.1万円
給与500万円 + 副業所得100万円: 上限約9.8万円
→ 約3.7万円分多く返礼品を受け取れる
```

:::message alert
ふるさと納税は自治体数が5箇所以下ならワンストップ特例が使えますが、確定申告をする場合は申告書に記載が必要です。
:::

### 6. 家事按分の適正化

自宅を仕事場にしている場合、以下を按分できます:

**按分の考え方:**

```python
# Pythonで按分計算のシミュレーション

def calculate_expense_ratio(work_space_sqm, total_space_sqm, work_hours_per_day):
    """
    経費按分率を計算
    """
    space_ratio = work_space_sqm / total_space_sqm
    time_ratio = work_hours_per_day / 24
    
    # 空間比率と時間比率の平均を取る方法
    expense_ratio = (space_ratio + time_ratio) / 2
    
    return min(expense_ratio, 0.5)  # 上限50%程度が安全

# 使用例
work_space = 10  # 作業スペース10㎡
total_space = 50  # 家全体50㎡
work_hours = 4  # 1日4時間作業

ratio = calculate_expense_ratio(work_space, total_space, work_hours)
print(f"経費按分率: {ratio:.1%}")
# 出力: 経費按分率: 27.1%

monthly_rent = 100000  # 家賃10万円
deductible_expense = monthly_rent * ratio
print(f"月間経費計上額: {deductible_expense:,.0f}円")
# 出力: 月間経費計上額: 27,083円
```

**按分が認められやすいポイント:**
- 作業専用の部屋・スペースがある
- 作業時間を記録している
- 業務使用の合理的な説明ができる

## 確定申告のスケジュール管理

計画的に進めることで、慌てずに対応できます。

### 年間スケジュール

```
1月
- 前年の収支確認
- 不足している領収書の確認

2月上旬
- 必要書類の収集完了
- 会計ソフトでの入力完了
- 申告書の下書き作成

2月中旬〜下旬
- 申告書の最終確認
- e-Tax送信 or 税務署提出

3月15日
- 確定申告期限

3月中
- 次年度の計画立案
- 必要に応じて青色申告承認申請
```

### チェックリスト

申告前に確認すべき項目:

- [ ] 全ての収入源を把握している
- [ ] 経費の領収書・レシートを整理している
- [ ] 按分計算の根拠を説明できる
- [ ] 控除書類(保険料控除証明書など)を揃えている
- [ ] マイナンバーカードまたは通知カードがある
- [ ] 還付先の口座情報を確認している

## よくある失敗例と対策

### 失敗例1: 経費を過剰に計上してしまう

**NGな例:**
- プライベートの飲食費を全額経費にする
- 業務関連性が薄い購入を経費化
- 按分が不適切(家賃100%など)

**対策:**
- 業務との関連性を明確に説明できるもののみ計上
- 按分は保守的に(50%以下が安全)
- 記録を残す(いつ・何のために使ったか)

### 失敗例2: 申告を後回しにして期限を過ぎる

**リスク:**
- 無申告加算税(15〜20%)
- 延滞税(年7.3〜14.6%)
- 青色申告の承認取消の可能性

**対策:**
- 毎月の記帳を習慣化
- 2月初旬には申告準備完了を目標に
- 不明点は早めに税務署に相談

### 失敗例3: 雑所得と事業所得を誤解する

2022年の税制改正で、収入300万円以下は原則雑所得とされやすくなりました。

**対策:**
- 帳簿書類をしっかり作成・保存
- 継続性・反復性を示す記録を残す
- 必要に応じて税理士に相談
