## 6. SQL・データ分析クエリの生成

### 課題
複雑なSQLクエリを書くのに時間がかかる。

### 解決策
自然言語で要件を伝え、SQLを生成してもらう。

### 具体的プロンプト

```
以下の要件でSQLクエリを作成してください：

【データベース構造】
- users テーブル（id, name, created_at）
- orders テーブル（id, user_id, amount, order_date）

【取得したいデータ】
2024年に登録したユーザーで、累計購入金額が10万円以上の人を、
購入金額の多い順に表示

【必要なカラム】
ユーザー名、累計購入金額、注文回数
```

### 生成されるSQLの例

```sql
SELECT 
    u.name,
    SUM(o.amount) as total_amount,
    COUNT(o.id) as order_count
FROM users u
INNER JOIN orders o ON u.id = o.user_id
WHERE u.created_at >= '2024-01-01'
GROUP BY u.id, u.name
HAVING SUM(o.amount) >= 100000
ORDER BY total_amount DESC;
```
