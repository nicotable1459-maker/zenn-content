## Custom Instructionsとは?

Custom Instructionsは、ChatGPTに対して**あなた専用の前提条件を設定できる機能**です。

### 従来の使い方(非効率)

```
毎回こんな指示を繰り返していませんか?

「あなたはシニアエンジニアです。コードレビューをしてください。
指摘はTypeScriptの型安全性、パフォーマンス、可読性の観点で...」
```

### Custom Instructions導入後(効率的)

```
設定を一度保存すれば、以降は:

「このコードをレビューして」

だけでOK!
```

設定した前提条件が自動的に適用され、毎回の指示が不要になります。

## Custom Instructionsの設定方法

### ステップ1:設定画面を開く

1. ChatGPT画面右下のアカウントアイコンをクリック
2. 「Settings」→「Personalization」→「Custom Instructions」を選択
3. 2つの入力欄が表示されます

### ステップ2:2つの質問に答える

Custom Instructionsは2つのセクションに分かれています:

**1. What would you like ChatGPT to know about you?**
(ChatGPTにあなたについて何を知ってほしいですか?)

→ あなたの背景、役割、専門分野などを記述

**2. How would you like ChatGPT to respond?**
(ChatGPTにどのように応答してほしいですか?)

→ 出力形式、トーン、制約条件などを指定

## Custom Instructions活用の5つのコツ

### 1. 具体的に書く

❌ **悪い例**
```
- プログラマーです
- 分かりやすく教えてください
```

✅ **良い例**
```
- バックエンドエンジニア(Python/Django、経験3年)
- マイクロサービスアーキテクチャの設計経験あり
- 現在REST APIのパフォーマンス改善に取り組み中
```

### 2. 出力形式を明確に指定

```
【推奨される指定例】

- コードブロックは言語名を必ず指定
- 説明は「結論→理由→具体例」の順
- 選択肢がある場合は比較表形式
- コード例は30行以内
- 専門用語には初出時に注釈
```

### 3. 制約条件を設定

```
【時短につながる制約】

- 前置きは不要、直接本題から
- 「それでは」「まず」などの接続詞を省略
- 繰り返しの確認は不要
- 要約は3行以内
```

### 4. 定期的に見直す

プロジェクトや役割が変わったら更新しましょう。

```
月1回のレビュー:
□ 使用技術スタックに変更はないか?
□ 新しい興味分野が増えたか?
□ 出力形式で改善点はないか?
□ 不要な指示が含まれていないか?
```

### 5. 用途別に使い分ける(有料版のみ)

ChatGPT Plusユーザーは、複数のカスタムGPTsを作成可能。

```
例:
- 「コードレビューGPT」→ 厳格な指摘
- 「ブレストGPT」→ 自由な発想
- 「文章校正GPT」→ 表現の洗練
```

## トラブルシューティング

### Q1. 設定が反映されない?

**確認ポイント:**
- 設定後に「Save」ボタンを押したか
- ブラウザのキャッシュクリア
- 新しい会話を開始しているか(既存の会話には適用されない)

### Q2. 出力が冗長になった

```
セクション2に追加:

- 説明は3文以内
- コード例優先、長文の説明は避ける
- 「簡潔に」と明記
```

### Q3. 専門性が下がった?

```
セクション1に追加:

- 専門用語を積極的に使用してOK
- 初心者向けの説明は不要
- 高度な技術的議論を期待
```
