## 1. メール文面の自動生成・添削

### 実装例：お詫びメール生成

ビジネスメールの作成は時間がかかる上に、表現に悩むことも多いタスクです。特にお詫びメールやお礼メールなど、定型的だけど丁寧さが求められる場面で威力を発揮します。

```python
import openai

def generate_apology_email(situation, recipient):
    prompt = f"""
以下の状況に対するお詫びメールを日本語で作成してください。
- 宛先: {recipient}
- 状況: {situation}

条件:
- ビジネスメールとして適切な敬語を使用
- 簡潔で誠意が伝わる内容
- 具体的な改善策を含める
"""
    
    response = openai.ChatCompletion.create(
        model="gpt-4o-mini",
        messages=[
            {"role": "system", "content": "あなたは経験豊富なビジネスライターです。"},
            {"role": "user", "content": prompt}
        ],
        temperature=0.7
    )
    
    return response.choices[0].message.content

# 使用例
email = generate_apology_email(
    situation="納期遅延により、プロジェクトに2日の遅れが発生",
    recipient="田中様"
)
print(email)
```

**実務での活用ポイント：**
- `temperature`を0.7程度に設定することで、適度な表現の多様性を確保
- システムプロンプトで役割を明確に指定することで、文体の一貫性を保つ
- 生成後は必ず人間がレビューし、固有名詞や数値を確認する

## ベストプラクティスとコスト最適化

### トークン使用量の削減

```python
def count_tokens(text, model="gpt-4o"):
    """トークン数を概算（正確にはtiktokenライブラリを使用）"""
    # 日本語の場合、1文字≒1.5〜2トークンが目安
    estimated_tokens = len(text) * 1.5
    return estimated_tokens

def optimize_prompt(long_text, max_tokens=1000):
    """プロンプトを最適化してトークン数を削減"""
    if count_tokens(long_text) > max_tokens:
        # 要約してからAPI呼び出し
        summary = summarize_text(long_text)
        return summary
    return long_text
```

### エラーハンドリングとリトライ戦略

```python
import time
from openai import OpenAI

client = OpenAI()

def api_call_with_retry(prompt, max_retries=3):
    for attempt in range(max_retries):
        try:
            response = client.chat.completions.create(
                model="gpt-4o-mini",
                messages=[{"role": "user", "content": prompt}],
                timeout=30
            )
            return response.choices[0].message.content
        
        except Exception as e:
            if attempt == max_retries - 1:
                raise
            wait_time = 2 ** attempt  # 指数バックオフ
            print(f"リトライ {attempt + 1}/{max_retries} - {wait_time}秒待機")
            time.sleep(wait_time)
```

### セキュリティとプライバシー

**重要な注意事項：**

1. **機密情報の取り扱い**

   - 社内データはオンプレミスLLMの使用を検討

2. **プロンプトインジェクション対策**
```python
def sanitize_user_input(user_input):
    """ユーザー入力をサニタイズ"""
    # 潜在的に危険な命令文を除去
    dangerous_patterns = [
        "ignore previous instructions",
        "new instructions:",
        "システムプロンプトを無視"
    ]
    
    sanitized = user_input
    for pattern in dangerous_patterns:
        sanitized = sanitized.replace(pattern, "")
    
    return sanitized
```

3. **出力の検証**
   - 重要な業務では必ず人間がレビュー
   - 自動判断が必要な場合は信頼度スコアを実装
