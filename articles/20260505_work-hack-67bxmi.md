## はじめに:Notionのポテンシャルを引き出す

Notionを使っているけど、なんだか作業が遅い...そう感じたことはありませんか?

実は、Notionには**マウス操作だけでは気づけない強力なショートカット機能**が隠されています。これらを習得することで、あなたの作業速度は文字通り2倍になります。

この記事では、私が2年以上Notionを使い倒して厳選した「本当に使える」ショートカット10選を紹介します。

**この記事で得られる価値:**
- Notionの作業速度が劇的に向上する
- マウスとキーボードの往復が減り、集中力が途切れにくくなる
- 知らなかった便利機能に出会える

それでは早速見ていきましょう!

## 1. ページ内検索:`Cmd/Ctrl + F`の真価

### 基本の使い方
多くの人が知っている基本中の基本ですが、Notionでは特に威力を発揮します。

```
Mac: Cmd + F
Windows: Ctrl + F
```

### 応用テクニック
長大なドキュメントやデータベース内で特定の情報を探す際、**検索→Enter→ESCの3ステップ**を習慣化すると、スクロール地獄から解放されます。

**実践例:**
週次レビュー文書で「先週の課題」を探す場合
1. `Cmd + F`で検索ボックス起動
2. 「課題」と入力
3. `Enter`で該当箇所にジャンプ
4. `ESC`で検索を閉じて編集開始

これだけで10秒以上の時短になります。

## 2. クイックファインド:`Cmd/Ctrl + P`で全てのページにアクセス

### なぜこれが最強なのか
サイドバーから目的のページを探す時代は終わりました。

```
Mac: Cmd + P
Windows: Ctrl + P
```

このショートカットを使えば、**ワークスペース内の全ページに瞬時にアクセス**できます。

### 実践的な使い方
1. `Cmd + P`を押す
2. ページ名の一部を入力(完全一致不要)
3. `↑↓`キーで選択
4. `Enter`で開く

**時短効果:**
サイドバーをクリックして探す場合: 約15秒
クイックファインド使用: 約3秒
→ **1日20回使えば4分の時短!**

## 3. 新規ページ作成:`Cmd/Ctrl + N`

### 思考の流れを止めない
アイデアが浮かんだ瞬間にページを作る。これが重要です。

```
Mac: Cmd + N
Windows: Ctrl + N
```

### 組み合わせ技
```
Cmd + N (新規ページ作成)
→ タイトル入力
→ Enter (本文に移動)
→ すぐに執筆開始
```

この流れを3秒以内に完了できるようになると、**アイデアの取りこぼしが激減**します。

## 4. スラッシュコマンド:`/`で魔法を発動

### Notionの真骨頂
`/`(スラッシュ)を入力すると、あらゆるブロックタイプにアクセスできます。

**頻出パターン:**
```
/h1 → 見出し1
/h2 → 見出し2
/todo → TODOリスト
/code → コードブロック
/table → テーブル
/callout → コールアウト
/toggle → トグルリスト
```

### 爆速入力テクニック
完全に入力する必要はありません!

```
/h → 見出し候補が表示
/to → TODOリスト、トグルが表示
/ta → テーブルが表示
```

最初の2〜3文字で目的のブロックが出現します。

### 実践例:議事録作成
```
タイトル入力
/h2 → 「参加者」
参加者リスト入力
/h2 → 「議題」
/todo → アクションアイテム入力
/table → 決定事項の整理
```

この流れをマウスなしで完遂できれば、**議事録作成時間が半分**になります。

## 5. テキスト装飾:`Cmd/Ctrl + B/I/U`

### マークダウンより速い
Notionはマークダウン記法も使えますが、ショートカットキーの方が直感的です。

```
太字: Cmd/Ctrl + B
斜体: Cmd/Ctrl + I
下線: Cmd/Ctrl + U
```

### さらに便利な装飾系
```
打ち消し線: Cmd/Ctrl + Shift + S
インラインコード: Cmd/Ctrl + E
```

**実践パターン:**
文章を書いてから、重要な部分をドラッグして`Cmd + B`。
これで**後から装飾できる**ので、執筆の流れが止まりません。

## 6. リンク挿入:`Cmd/Ctrl + K`

### URLを美しく埋め込む
URLをそのまま貼るのではなく、テキストにリンクを埋め込みましょう。

```
Mac: Cmd + K
Windows: Ctrl + K
```

**手順:**
1. リンクを付けたいテキストを選択
2. `Cmd + K`
3. URLを貼り付け
4. `Enter`

これで「[参考記事](URL)」のような美しいリンクが完成します。

### 時短効果
マウスで右クリック→「リンク」選択: 約5秒
ショートカット使用: 約2秒

## 7. ブロック移動:`Cmd/Ctrl + Shift + ↑↓`

### 並び替えが超高速に
ブロック(段落、画像、テーブルなど)をドラッグする必要はありません。

```
上に移動: Cmd/Ctrl + Shift + ↑
下に移動: Cmd/Ctrl + Shift + ↓
```

### 実践例:記事の構成変更
```
見出し3の上にカーソル
→ Cmd + Shift + ↑を5回
→ 見出し2の上に移動完了
```

**マウス操作との比較:**
- マウスドラッグ: 画面スクロールしながら移動→約10秒
- ショートカット: 矢印キー連打→約3秒

## 8. コメント機能:`Cmd/Ctrl + Shift + M`

### チーム作業の効率化
テキストを選択して、

```
Mac: Cmd + Shift + M
Windows: Ctrl + Shift + M
```

これで瞬時にコメントを追加できます。

### 使用シーン
- レビュー依頼
- 疑問点の明示
- 補足説明の追加

**従来の方法:**
マウスでテキスト選択→右クリック→「コメント」選択: 約6秒

**ショートカット:**
テキスト選択→`Cmd + Shift + M`: 約2秒

## 9. 複製:`Cmd/Ctrl + D`

### テンプレート活用の要
ブロックやページを複製する最速の方法。

```
Mac: Cmd + D
Windows: Ctrl + D
```

### 実践例:週次レポート
```
先週のレポートページを開く
→ Cmd + D (ページ全体を複製)
→ 日付とデータを更新
→ 完成!
```

フォーマットを一から作る必要がなくなります。

### 応用:連続複製
```
TODOリストのテンプレートをクリック
→ Cmd + D を3回連打
→ 3つのTODOブロックが生成
```

## 10. 元に戻す/やり直し:`Cmd/Ctrl + Z/Shift + Z`

### 最後の砦
どのアプリでも基本ですが、Notionでは特に重要。

```
元に戻す: Cmd/Ctrl + Z
やり直し: Cmd/Ctrl + Shift + Z
```

### Notionの優れた点
Notionの履歴機能は**非常に細かく記録**されているので、
- 誤ってブロックを削除
- 大規模な編集を元に戻したい
- 過去のバージョンを確認したい

といった場合でも、`Cmd + Z`を連打すれば大抵解決します。

**Pro Tip:** 大胆な編集前に`Cmd + D`でページを複製しておけば、さらに安心です。

## ボーナス:組み合わせ技で真価を発揮

```
→ Cmd + Shift + M → コメント追加

```
所要時間:**約15秒**(マウス操作なら1分以上)
