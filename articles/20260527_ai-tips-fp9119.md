## 1. メール返信の自動下書き生成

### 実装概要

受信メールの内容を解析し、適切な返信文を自動生成します。忙しい朝のメール対応時間を大幅に短縮できます。

### コード例

```python
import openai
import os

openai.api_key = os.getenv("OPENAI_API_KEY")

def generate_email_reply(received_email):
    """受信メールから返信文を生成"""
    
    prompt = f"""
以下のメールに対して、丁寧でプロフェッショナルな返信文を作成してください。

【受信メール】
{received_email}

【要件】
- ビジネスメールとして適切な敬語を使用
- 具体的な回答内容を含める
- 署名は「[署名]」として残す
"""
    
    response = openai.chat.completions.create(
        model="gpt-4o-mini",  # コスト重視
        messages=[
            {"role": "system", "content": "あなたは経験豊富なビジネスパーソンです。"},
            {"role": "user", "content": prompt}
        ],
        temperature=0.7,
        max_tokens=500
    )
    
    return response.choices[0].message.content

# 使用例
email_content = """
田中様

お世話になっております。
来週の打ち合わせの件ですが、14日の15時からでご都合いかがでしょうか?
場所は弊社会議室を予定しております。

よろしくお願いいたします。
"""

reply = generate_email_reply(email_content)
print(reply)
```

### 効果

- **時短効果**: 1通あたり5分 → 30秒（約90%削減）
- **月間コスト**: 約$2〜3（1日20通処理の場合）

## 4. ドキュメント翻訳の一括処理

### 実装概要

技術文書やマニュアルを複数言語に一括翻訳します。専門用語の一貫性を保ちながら翻訳可能です。

### コード例

```python
def translate_document(text, target_lang="日本語", glossary=None):
    """ドキュメント翻訳（用語集対応）"""
    
    glossary_text = ""
    if glossary:
        glossary_text = "\n【用語集】\n" + "\n".join(
            [f"- {k}: {v}" for k, v in glossary.items()]
        )
    
    prompt = f"""
以下の技術文書を{target_lang}に翻訳してください。

【要件】
- 技術的正確性を保つ
- 自然な{target_lang}表現を使用
- 用語集がある場合は必ず従う

{glossary_text}

【原文】
{text}
"""
    
    response = openai.chat.completions.create(
        model="gpt-4o",
        messages=[
            {"role": "user", "content": prompt}
        ],
        temperature=0.3
    )
    
    return response.choices[0].message.content

# 使用例
glossary = {
    "API": "API",
    "endpoint": "エンドポイント",
    "authentication": "認証"
}

doc = "This API endpoint requires authentication using OAuth 2.0."
translated = translate_document(doc, glossary=glossary)
print(translated)
```

```python
【FAQ】
{faq_text}

{question}

【回答要件】
- FAQの内容を基に回答
- 丁寧で分かりやすい説明
- 該当するFAQがない場合は、その旨を伝える

    response = openai.chat.completions.create(
        model="gpt-4o-mini",
        messages=[
            {"role": "system", "content": "あなたは親切なカスタマーサポート担当です。"},
            {"role": "user", "content": prompt}

        temperature=0.3
    )
```

## 7. エラーハンドリングとリトライ機能の実装

### 実装概要

API呼び出しの信頼性を高めるため、適切なエラーハンドリングとリトライ機能を実装します。

### コード例

```python
import time
from openai import OpenAI, APIError, RateLimitError, APIConnectionError

def call_api_with_retry(prompt, max_retries=3):
    """リトライ機能付きAPI呼び出し"""
    
    client = OpenAI()
    
    for attempt in range(max_retries):
        try:
            response = client.chat.completions.create(
                model="gpt-4o-mini",
                messages=[{"role": "user", "content": prompt}],
                timeout=30
            )
            return response.choices[0].message.content
            
        except RateLimitError:
            if attempt < max_retries - 1:
                wait_time = (2 ** attempt) * 2  # 指数バックオフ
                print(f"Rate limit reached. Waiting {wait_time}s...")
                time.sleep(wait_time)
            else:
                raise
                
        except APIConnectionError:
            if attempt < max_retries - 1:
                print(f"Connection error. Retry {attempt + 1}/{max_retries}")
                time.sleep(2)
            else:
                raise
                
        except APIError as e:
            print(f"API Error: {e}")
            raise
    
    raise Exception("Max retries exceeded")

# 使用例
try:
    result = call_api_with_retry("こんにちは、ChatGPT!")
    print(result)
except Exception as e:
    print(f"Failed after retries: {e}")
```

```python
def monitor_token_usage(response):
    """トークン使用量の記録"""
    usage = response.usage
    cost_per_1k_input = 0.00015  # gpt-4o-miniの場合
    cost_per_1k_output = 0.0006
    
    input_cost = (usage.prompt_tokens / 1000) * cost_per_1k_input
    output_cost = (usage.completion_tokens / 1000) * cost_per_1k_output
    total_cost = input_cost + output_cost
    
    print(f"Tokens: {usage.total_tokens} | Cost: ${total_cost:.4f}")
    
    return total_cost
```
