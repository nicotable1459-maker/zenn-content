## はじめに:副業で稼ぎ始めたあなたへ

副業を始めて収入が増えてくると嬉しい反面、「税金ってどうなるの?」「確定申告って必要?」という疑問が湧いてきますよね。

実は、**副業所得が年間20万円を超えると確定申告が必要**になります。月5万円なら年間60万円。完全に申告対象です。

この記事では、副業で稼ぎ始めた方が必ず知っておくべき税金の基礎知識と、実際の確定申告の流れ、さらに合法的に節税する方法まで網羅的に解説します。

**この記事を読むとわかること:**
- 副業収入にかかる税金の種類と計算方法
- 確定申告が必要になるケースと具体的な手順
- 経費として認められる項目と記帳方法
- 青色申告と白色申告の違いと選び方
- 実際に使える節税テクニック

## 副業収入にかかる税金の基本

### 所得税の仕組み

副業で得た収入は「雑所得」または「事業所得」として扱われ、所得税の対象になります。

**所得税の計算式:**
```
所得税 = (総収入 - 必要経費 - 各種控除) × 税率
```

税率は累進課税で、所得が増えるほど税率も上がります:

| 課税所得金額 | 税率 | 控除額 |
|------------|------|--------|
| 195万円以下 | 5% | 0円 |
| 195万円超〜330万円以下 | 10% | 97,500円 |
| 330万円超〜695万円以下 | 20% | 427,500円 |
| 695万円超〜900万円以下 | 23% | 636,000円 |

### 住民税も忘れずに

所得税だけでなく、**住民税(約10%)**も課税されます。こちらは翌年6月から納付が始まります。

### 具体的なシミュレーション

**ケース:副業で年間60万円稼いだ場合**

```
副業収入: 600,000円
必要経費: 100,000円(PC代、通信費など)
副業所得: 500,000円

本業の給与所得: 4,000,000円
合計所得: 4,500,000円

所得控除(基礎控除等): 1,000,000円
課税所得: 3,500,000円

所得税: 3,500,000円 × 20% - 427,500円 = 272,500円
住民税: 3,500,000円 × 10% = 350,000円
```

副業をしない場合と比較すると、**約10万円の税負担増**となります。

## 確定申告が必要になるケース

### 絶対に申告が必要な場合

1. **給与所得者で副業所得が20万円超**
2. **給与所得が2,000万円超**
3. **2カ所以上から給与を受けている**
4. **医療費控除などを受けたい場合**

### 20万円以下でも申告すべきケース

実は20万円以下でも申告したほうが得な場合があります:

- **源泉徴収されている場合**(還付される可能性)
- **赤字を翌年に繰り越したい場合**(青色申告)
- **住民税は20万円ルールが適用されない**ため、市区町村への申告は必要

## 経費として認められる項目と記帳方法

### 副業で経費にできる主な項目

**物理的な支出:**
- パソコン・タブレット・スマートフォン
- ソフトウェア・アプリの購入費
- 書籍・教材費
- セミナー参加費

**継続的な支出:**
- インターネット回線費用
- クラウドサービス利用料(AWS、Notionなど)
- レンタルサーバー代
- ドメイン代

**間接的な支出:**
- 自宅作業スペースの家賃(按分)
- 電気代(按分)
- 打ち合わせの飲食代
- 交通費

### 按分の考え方

自宅で副業をする場合、家賃や光熱費を**按分**して経費にできます。

```
例:家賃10万円のマンションで、専有面積の20%を作業スペースとして使用

副業での使用時間: 1日3時間(全体の12.5%)

経費算入可能額 = 100,000円 × 20% × 12.5% = 2,500円/月
年間: 30,000円
```

合理的な根拠があれば認められますが、**過度な按分は税務調査で指摘される**可能性があります。

### 帳簿の付け方

**最低限必要な記録:**
- 日付
- 取引内容
- 金額
- 取引先

**簡単な記帳例(スプレッドシート):**

| 日付 | 内容 | 収入 | 支出 | 項目 | 残高 |
|------|------|------|------|------|------|
| 2024/01/15 | クライアントA 記事執筆料 | 50,000 | - | 売上 | 50,000 |
| 2024/01/20 | ChatGPT Plus | - | 2,980 | 通信費 | 47,020 |
| 2024/01/25 | 技術書購入 | - | 3,500 | 新聞図書費 | 43,520 |

**おすすめツール:**
- freee
- マネーフォワード クラウド確定申告
- やよいの青色申告オンライン

これらは銀行口座やクレカと連携でき、自動で帳簿を作成してくれます。

## 青色申告 vs 白色申告

### 白色申告の特徴

**メリット:**
- 帳簿が簡易でOK
- 事前の届出不要

**デメリット:**
- 特別控除なし
- 赤字の繰越不可

### 青色申告の特徴

**メリット:**
- **最大65万円の特別控除**(e-Tax使用時)
- 赤字を3年間繰り越せる
- 家族への給与を経費にできる
- 30万円未満の資産を一括償却可能

**デメリット:**
- 複式簿記が必要(ツール使えば簡単)
- 事前の届出が必要(開業から2ヶ月以内)

### どちらを選ぶべきか

**副業所得が年間50万円以上なら青色申告一択**です。

```
シミュレーション:副業所得100万円の場合

【白色申告】
課税所得: 1,000,000円
所得税(税率10%): 100,000円

【青色申告(65万円控除)】
課税所得: 1,000,000円 - 650,000円 = 350,000円
所得税(税率5%): 17,500円

差額: 82,500円の節税!
```

## 確定申告の具体的な手順

### 1. 必要書類の準備(1月中)

- 源泉徴収票(本業)
- 副業の収入・支出の記録
- 各種控除証明書(生命保険、ふるさと納税など)
- マイナンバーカード

### 2. 申告書の作成(2月初旬〜)

**国税庁の「確定申告書等作成コーナー」を使う:**

https://www.keisan.nta.go.jp/

画面の指示に従って入力するだけで、自動計算してくれます。

**入力の流れ:**
```
1. 給与所得の入力(源泉徴収票を見ながら)
2. 雑所得または事業所得の入力
   - 収入金額を入力
   - 必要経費を入力
3. 所得控除の入力
   - 社会保険料控除
   - 生命保険料控除
   - 寄附金控除(ふるさと納税)
4. 税額の自動計算
5. 還付または納付額の確定
```

### 3. 提出(3月15日まで)

**提出方法は3つ:**

1. **e-Tax(オンライン)** ← おすすめ!
   - 自宅から提出可能
   - 青色申告特別控除65万円が使える
   - マイナンバーカードが必要

2. **郵送**
   - 税務署に郵送
   - 控えが欲しい場合は返信用封筒を同封

3. **税務署に持参**
   - 混雑するので避けたい

### 4. 納税または還付(3月15日まで)

- 納税が必要な場合:口座振替、クレジットカード、コンビニ払い
- 還付の場合:1〜2ヶ月後に指定口座に振込

## 実践的な節税テクニック

### 1. iDeCoを活用する

年間最大27.6万円まで拠出でき、**全額が所得控除**の対象になります。

```
年収500万円、iDeCo拠出額27.6万円の場合

節税額: 276,000円 × 20%(所得税率) = 55,200円
       276,000円 × 10%(住民税率) = 27,600円
合計: 82,800円の節税
```

### 2. ふるさと納税を最大限活用

副業で所得が増えると、ふるさと納税の上限額も増えます。

```
シミュレーションサイトで上限額をチェック:
給与所得400万円 + 副業所得50万円 = 450万円
→ 上限額約52,000円

実質2,000円で50,000円分の返礼品がもらえる!
```

### 3. 小規模企業共済

個人事業主として開業届を出せば加入可能。掛金は全額所得控除で、退職金代わりになります。

### 4. 経費の計上漏れをなくす

意外と忘れがちな経費:
- スマホ代(按分)
- 自宅の一部を事務所として使用(家賃按分)
- 副業関連の飲食代
- セミナー参加の交通費・宿泊費

### 5. クレジットカードを分ける

**副業専用のクレジットカードを作ると:**
- 経費の管理が楽になる
- 会計ソフトとの連携が簡単
- プライベートとの区別が明確

おすすめは年会費無料の法人カード(個人事業主でも作れる):
- 三井住友カード ビジネスオーナーズ
- freeeカード

## よくある失敗と注意点

### 失敗1:確定申告を忘れて延滞税

期限を過ぎると**無申告加算税(15〜20%)**と**延滞税(年7.3〜14.6%)**がかかります。

### 失敗2:経費の領収書を捨てる

原則として**7年間の保管義務**があります。デジタル化してクラウド保存がおすすめ。

### 失敗3:副業が会社にバレる

住民税の徴収方法を「特別徴収(給与天引き)」にすると会社にバレる可能性があります。

**対策:**
確定申告書の「住民税の徴収方法」で**「自分で納付」**を選択する。

### 失敗4:架空経費の計上

当然ですが、実際にない経費を計上するのは脱税です。税務調査が入ると追徴課税されます。

## まとめ:副業で賢く稼ぐために

副業で月5万円以上稼げるようになったら、税金の知識は必須です。

**この記事の重要ポイント:**

1. **副業所得20万円超で確定申告が必要**
2. **青色申告で最大65万円の控除**を活用
3. **経費をしっかり計上**して課税所得を減らす
4. **iDeCoやふるさと納税**で合法的に節税
5. **会計ソフト**を使えば帳簿付けは簡単
6. **期限厳守**(3月15日)で延滞税を回避

最初は複雑に感じるかもしれませんが、一度流れを理解すれば翌年からはスムーズです。

むしろ確定申告を通じて、自分の収支を見える化することで、副業をビジネスとして成長させるヒントも得られます。
