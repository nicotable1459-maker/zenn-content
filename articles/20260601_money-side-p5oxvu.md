## なぜ月5万円なのか？

月5万円という金額には明確な理由があります。

### 生活に与えるインパクト

- **年間60万円**の追加収入
- 投資に回せば複利効果で将来の資産形成に大きく貢献
- 趣味や自己投資に使える余裕が生まれる
- 精神的な安定感（収入源の複数化）

### 現実的な達成可能性

月5万円は、本業を持つエンジニアが無理なく達成できる金額です。月30万円、50万円といった大きな目標は魅力的ですが、最初のステップとしては負担が大きすぎます。

### 税金の基礎知識

**確定申告が必要になる条件：**
```
副業の所得が年間20万円を超える場合
※所得 = 収入 - 経費

例）
年間収入: 60万円
経費: 5万円（PC代、書籍代、通信費等）
所得: 55万円 → 確定申告が必要
```

**経費として計上できるもの：**
- PC、周辺機器（按分が必要な場合あり）
- 技術書籍、オンライン教材
- インターネット回線費用（按分）
- コワーキングスペース利用料
- セミナー参加費

**税金の計算例：**
```python
# 簡易計算（実際は所得税率による）
副業所得 = 600000  # 年間60万円
経費 = 50000
課税所得 = 副業所得 - 経費

# 所得税（税率10%と仮定）
所得税 = (課税所得 * 0.10)
# 住民税（約10%）
住民税 = (課税所得 * 0.10)

合計税額 = 所得税 + 住民税
実質手取り = 副業所得 - 経費 - 合計税額

print(f"実質手取り: {実質手取り:,}円")
# 出力: 実質手取り: 440,000円
```
